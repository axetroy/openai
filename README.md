## open.ai SDK

open.ai 的 Golang 非官方 SDK 以及 chatGPT 的命令行工具

### 安装

```bash
go get -u github.com/axetroy/openai
```

### 命令行工具

- [chatGPT](cmd/chatgpt/README.md)

### 测试

```bash
$ make test
```
